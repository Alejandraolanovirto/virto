File Structure

A guide to understanding how the theme is structured. Within this theme, you‚Äôll find the following directories and files, grouping common resources and providing both compiled and minified distribution files, as well as raw source files.

Page Templates

üìÅ app is the root directory and source for all HTML pages (in .html or .njk) and contains the example HTML templates supplied with the theme.

Contained in app directory/folder is a precompiled static site or sites, equivalent to what you see in the online demo.

Pages are named generally according to their demo name, number or purpose, and you can open any of the files in your browser to see it in action, or modify the markup in your editor to suit your needs.

As this is the source for all files, Gulp copies all files from the app to the dist folder/directory and does necessary compilation during development.

This documentation is also housed in app to share the theme's assets, but the documentation files are not required in your projects, and you may move or make another copy of them elsewhere. These docs are also available [online](http://grayic.com/documentation/).

Folder Contents

üìÅ app contains fonts, image, js, partials, plugins, scss, and other HTML files.

  • üìÅ fonts contains all third party fonts/icons used in the theme.

  • üìÅ image contains all images used in this theme, backgrounds, icons, patterns and more.

  • üìÅ js contains custom js codes required for theme functionality.

  • üìÅ partials contains all required common parts, blocks, chunk of code used in each theme (e.g. header, navigation menu, template, sidebar, etc.) in nunjucks files(.njk)

  • üìÅ plugins contains both required and additional plugins (with all their contents e.g. .js, and .css files)

  • üìÅ scss contains all necessary SCSS for theme stylings (other than plugin CSS files)

SCSS

üìÅ scss contains the source for all theme CSS. You will find the following contents in scss:

üìÅ base, üìÅ components, üìÅ helpers, üìÅ pages, üìÅ sections, and custom, dark-versions, main and settings files.

  • üìÅ base contains common.scss and spacings.scss

    • common.scss contains all common stylings used throughout the theme.

    • spacings.scss contains all spacing classes and values.

  • üìÅ components contains common parts, blocks of each page such as header, footer, menu, etc.

  • üìÅ helper contains mixins and variables which helps to control consistent color, fonts throughout the theme and makes code cleaner, simpler.

    • mixin.scss contains scss mixins for media queries and screen sizes

    • variables.scss contains theme colors, font family options and any values that can be used as variables.

  • üìÅ pages contains individual demo page and inner page styles.

  • üìÅ sections contains individual sections and repeated block styles.

  • settings.scss contains the mandatory resets of html, body and bootstrap.

  • main.scss includes/links all theme and page styles in one place to compile into main.css file in dist/css directory.

  • custom.scss is a place for you to add your own styles to keep them separate from the core theme files, which aids updating.
