Edit the Color Scheme

You can easily transform the color scheme of your site using SCSS.

Editing the default color scheme variables

Editing the default color scheme variables will change the colors in main.css and result the theme colors to change. Follow below instructions for changing heading titles, paragraph texts, buttons and theme base colors.

  1. Open the variables.scss file

    The variables.scss file is located in the app/scss/helpers directory of the theme package.

  2. Change the variable values to your preferred colors

    Inside variables.scss, each landing or demo page has it's own section to specify their color scheme variables. Modify the HEX values to change the color scheme.

                

    // Theme Fonts and colors
    // =====================================
    $theme-primary-heading: #161c2d;
    $theme-primary-text: rgba(#161c2d, .6);
    $theme-primary-color: #473bf0;
    $theme-secondary-color: #68d585;

    // Landing1 Fonts and colors
    // =====================================
    $l1-primary-heading: #161c2d;
    $l1-primary-text: rgba(#161c2d, .6);
    $l1-primary-color: #473bf0;
    $l1-secondary-color: #68d585;
